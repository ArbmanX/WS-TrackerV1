# M4: Planners Hub

> Depends on: M2
> Unlocks: M5, M15

## Goal

Admin/manager view of all planners. Stat cards at top, card grid below for roster, coaching, trends, and regional comparison.

## Micro Tasks

- [ ] Stat cards row — Active planners, Behind Quota, Avg miles/week
- [ ] Planner Roster card — sortable/filterable list with status, quota, region → links to Planner Detail
- [ ] Coaching Signals card — planners needing attention ranked by urgency
- [ ] Career Trends card — team-wide trajectory chart
- [ ] Regional Comparison card — planners grouped by region with avg metrics
- [ ] Wire to PlannerMetricsService, PlannerCareerLedgerService, CoachingMessageGenerator
