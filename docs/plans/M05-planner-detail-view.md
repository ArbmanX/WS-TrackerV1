# M5: Planner Detail View

> Depends on: M4
> Unlocks: M14, M16

## Goal

Individual planner page at `/planners/{username}`. Full profile of a single planner's work, progress, and coaching.

## Micro Tasks

- [ ] Planner header — name, region, role, status badge
- [ ] Quota progress section — gauge/bar showing current vs target
- [ ] Career trend chart — historical weekly miles
- [ ] Assigned circuits list — cards per circuit with assessment counts and completion %
- [ ] Active assessments list — current work with status
- [ ] Coaching messages section — generated coaching text
- [ ] Recent activity feed — last actions/syncs for this planner
- [ ] Cross-links: circuit names → Circuit Detail, assessment GUIDs → Assessment Detail
