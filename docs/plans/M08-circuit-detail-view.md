# M8: Circuit Detail View

> Depends on: M6
> Unlocks: M9, M16

## Goal

Circuit page at `/assessments/circuit/{id}`. All assessments on a circuit with unit and permissioning breakdown.

## Micro Tasks

- [ ] Circuit header — name, region, assigned planner, status badge
- [ ] Assessment list — table/cards of assessments with status, extension, scope year
- [ ] Completion progress — overall % and breakdown by status
- [ ] Unit breakdown — work vs non-work unit counts
- [ ] Permissioning status — approved/pending/refused/deferred counts
- [ ] Cross-links: assessment GUIDs → Assessment Detail, planner name → Planner Detail, region → Region Detail
